## Quick Start

- Setup: Start a local server in the phaser-game-dodger folder.
- Play: Open http://localhost:8080 in your browser.
- Controls:
    Spacebar: Start game / Shoot
    Arrow Keys: Move your ship
- Goal: Dodge enemies, collect power-ups, and get the highest score.
